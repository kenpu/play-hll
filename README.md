# playhll

## Playing with HLL

This requires local installation of https://github.com/bigmlcom/sketchy

    git clone https://github.com/bigmlcom/sketchy ./sketch
    cd ./sketch
    lein deps
    lein install

Running some sample

    lein run -m playhll.datagen
